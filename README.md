# JASPEL-KPI System: Enterprise Incentive & Performance Management

Sistem manajemen insentif dan KPI modern yang dirancang untuk efisiensi tinggi, performa cepat, dan tampilan responsif.

## Fitur Utama
- **Dashboard Interaktif**: Visualisasi performa unit dan individu secara real-time.
- **Manajemen Unit & Pegawai**: Pengelolaan struktur organisasi dan data SDM.
- **Konfigurasi KPI**: Pengaturan indikator dan sub-indikator fleksibel.
- **Penilaian (Assessment)**: Sistem penilaian terintegrasi dengan validasi otomatis.
- **Export Laporan**: Dokumentasi profesional dalam format PDF (Slip Gaji) dan Excel (Ringkasan KPI).
- **Keamanan Lapis Tinggi**: Middleware teroptimasi dan Role-Based Access Control (RBAC).

## Teknologi
- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Visualisasi**: Recharts
- **Export**: jsPDF, SheetJS

## Memulai
```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev
```
